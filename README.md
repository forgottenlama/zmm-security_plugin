# ZMM security plugin
Je to plugn určený pre stránku `www.zmm.sk`. Jeho hlavnou úlohou je umožniť vytvárať stránky na ktoré je umožnený prístup až po prihlásení. Taktiež vytvára prihlasovacie údaje a kontroluje ich.
---
